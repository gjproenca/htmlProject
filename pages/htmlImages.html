Imagens

Estrutura de uma Imagem

Elementos

<align> Especifica o alinhamento de uma imagem de acordo com os elementos circundantes.
<alt> Especifica um texto alternativo para uma imagem.
<border> Especifica a largura da borda em torno de uma imagem.
<crossorigin> Permitir imagens de sites de terceiros que permitam o acesso de origem cruzada a ser usado com o canvas.
<height> Especifica a altura de uma imagem.
<hspace> Especifica o espaço em branco à esquerda e à direita de uma imagem.
<ismap> Especifica uma imagem como um mapa de imagem do lado do servidor.
<longdesc> Especifica um URL para uma descrição detalhada de uma imagem.
<sizes> Especifica tamanhos de imagem para diferentes layouts de página.
<src> Especifica o URL de uma imagem.
<srcset> Especifica o URL da imagem para usar em diferentes situações.
<usemap> Especifica uma imagem como um mapa de imagem do lado do cliente.
<vspace> Especifica o espaço em branco na parte superior e inferior de uma imagem.
<width> Especifica a largura de uma imagem.

Exemplo

<!DOCTYPE html>
<html>
<head>
<style>
img { 
    width:100%; 
}
</style>
</head>
<body>

<img src="html5.gif" alt="HTML5 Icon" width="128" height="128">
<img src="html5.gif" alt="HTML5 Icon"style="width:128px;height:128px;">

</body>
</html>
