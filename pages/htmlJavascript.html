Javascript

Estrutura Javascript num documento HTML

Elementos

<script> Definir o script do lado do cliente.
<noscript> Defina um conteúdo alternativo para usuários que não suportam scripts do lado do cliente.

Exemplo

<!DOCTYPE html>
<html>
<body>

<p id="demo"></p>

<script>
document.getElementById("demo").innerHTML = "Hello JavaScript!";
</script>

<noscript>Sorry, your browser does not support JavaScript!</noscript>

<p>A browser without support for JavaScript will show the text written inside the noscript element.</p>

</body>
</html>
