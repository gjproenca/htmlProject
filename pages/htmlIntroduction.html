Introdução

Estrutura de um documento Html

HTML significa Hyper Text Markup Language.
HTML descreve a estrutura das páginas da Web usando marcação.
Os elementos HTML são os blocos de construção de páginas HTML.
Elementos HTML são representados por tags.
Etiquetas HTML descrevem peças de conteúdo, como "título", "parágrafo", "tabela", e assim por diante.
Os navegadores não mostram as tags HTML, mas usam-nas para processar e renderizar o conteúdo da página.

Elementos

A declaração <!DOCTYPE html> define este documento como HTML5.
O elemento <head> contém informações meta sobre o documento.
O elemento <html> é o elemento raiz de uma página HTML.
O elemento <title> especifica um título para o documento.
O elemento <body> contém o conteúdo visível da página.
O elemento <h1> define um cabeçalho grande.
O elemento <p> define um parágrafo.

Estrutura de um documento HTML

<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>

<h1>My First Heading</h1>
<p>My first paragraph.</p>

</body>
</html>
