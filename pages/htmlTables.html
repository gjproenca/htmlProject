Tabelas

Estrutura de uma tabela

Elementos

Use o elemento HTML <table> para definir uma tabela.
Use o elemento HTML <tr> para definir uma linha da tabela.
Use o elemento HTML <td> para definir os dados de uma tabela.
Use o elemento HTML <th> para definir um cabeçalho de tabela.
Use o elemento HTML <caption> para definir uma legenda de tabela.
Use a propriedade CSS border para definir uma borda.
Use a propriedade CSS border-collapse para recolher as bordas das células.
Use a propriedade CSS padding x para adicionar preenchimento às células.
Use a propriedade CSS text-align x para alinhar o texto da célula.
Use a propriedade CSS border-spacing x para definir o espaçamento entre as células.
Use o atributo colspan para expandir uma célula em várias colunas.
Use o atributo rowspan para fazer com que uma célula se estenda por várias linhas.
Use o atributo id para definir o estilo de uma tabela em exclusivo.

Exemplo

<table style="width:100%">
  <caption>Monthly savings</caption>
  <tr>
    <th>Month</th>
    <th>Savings</th>
  </tr>
  <tr>
    <td>January</td>
    <td>$100</td>
  </tr>
  <tr>
    <td>February</td>
    <td>$50</td>
  </tr>
</table>
