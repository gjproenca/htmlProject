Css

Estrutura Css numa página HTML

Elementos

<style> Define informações de estilo para um documento HTML.
<link> Define um link entre um documento e um recurso externo.

Exemplo

<!DOCTYPE html>
<html>
<head>
<style>
h1 {
    color: blue;
    font-family: verdana;
    font-size: 300%;
}
p  {
    color: red;
    font-family: courier;
    font-size: 160%;
}
</style>
</head>
<body>

<h1>This is a heading</h1>
<p>This is a paragraph.</p>

</body>
</html>
