Projeto Final

Este site é o Projeto Final da UFCD, Programação em Cobol - Fundamentos.

Com este projeto pretende-se desenvolver a capacidade de trabalho autónomo e em equipa. O mesmo tem também o objetivo de ficar-mos a conhecer liguagens de programação viradas para a web.

Esta UFCD foi lecionada pelo formador José Sampaio e, teve a duração de 50 horas.

A matéria incidiu sobre a estrutura de uma página em HTML, CSS, Formulários, Validações em HTML 5/Javascript, criação de hyperlinks, botões, tabelas, barra de navegação com o uso de listas. Isto tudo, com a finalidade de termos os elementos necessários para conseguirmos criar um website.

Ao longo das aulas, foi-nos dada a matéria, seguida de exercícios práticos afim de podermos exercitar e ter uma melhor compreensão dos novos conteúdos a que fomos expostos.

A avaliação desta UFCD é composta por dois elementos de avaliação. O primeiro, a cotação de um teste em que o objetivo era fazer um site consistindo de algumas páginas mas com o objetivo de ter formulários e validações em funcionamento. O segundo elemento de avaliação, é a realização deste site onde, temos um nível de exigência mais alta, tendo o mesmo de ser composto por um mínino de dez páginas, três formulários e as respetivas validações em HTML5/Javascript. Este segundo elemento de avaliação terá de conter também a matéria dada em sala de aula.
